About OAuth

OAuth 2.0 is an authorization framework that enables applications to obtain limited access to user accounts on an HTTP service. It works over HTTPS and authorizes devices, APIs, servers, and applications with access tokens.

Common Flows:
  • Authorization Code: Most secure, for web apps
  • Implicit: For public clients (deprecated)
  • Client Credentials: For machine-to-machine
  • Resource Owner: For trusted applications

API Security
