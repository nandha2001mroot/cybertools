About security.txt

security.txt defines a standard to help organizations define the process for security researchers to securely disclose security vulnerabilities.

Required Fields:
  • Contact: Email address for security reports
Optional Fields:
  • Encryption: PGP key for encrypted communication
  • Signature: Signature for verification
  • Policy: URL to security policy
  • Acknowledgments: URL to hall of fame
