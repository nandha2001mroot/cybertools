About XSS

Cross-site scripting (XSS) is a security vulnerability that enables attackers to inject client-side scripts into web applications. XSS can be used to steal cookies, session tokens, or other sensitive information.

XSS Types:
  • Reflected XSS: Script reflected in response
  • Stored XSS: Script stored on server
  • DOM-based XSS: Script executes in DOM
  • Blind XSS: Script executes on different system
