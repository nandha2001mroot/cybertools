About robots.txt

robots.txt is a text file that webmasters create to instruct search engine robots how to crawl pages on their website. It's part of the robots exclusion protocol.

Common Directives:
  • User-agent: Specifies robots to which rules apply
  • Disallow: Specifies which URLs not to crawl
  • Allow: Specifies which URLs to crawl
  • Sitemap: Specifies location of sitemap
