About HTTP Cookies

                

                

                    
HTTP cookies are small pieces of data sent from a website and stored on the user's computer by the web browser. They are used to remember stateful information.

                    
Security Attributes:

                    
    
                        
  • HttpOnly: Prevents XSS attacks
    • 
                        
  • Secure: Ensures HTTPS transmission
    • 
                        
  • SameSite: Prevents CSRF attacks
    • 
                        
  • Domain: Specifies cookie scope
    • 
                        
  • Path: Limits cookie scope
