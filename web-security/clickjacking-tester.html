About Clickjacking

Clickjacking is a malicious technique of tricking a user into clicking something different from what the user perceives, potentially revealing confidential information or taking control of their computer.

Protection Methods:
  • X-Frame-Options header
  • Content Security Policy (CSP) frame-ancestors
  • Frame-busting JavaScript
  • Browser security features
