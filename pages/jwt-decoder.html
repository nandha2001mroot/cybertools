About JWT

JSON Web Token (JWT) is a compact, URL-safe means of representing claims to be transferred between two parties. JWTs are commonly used for:

  • Authentication
  • Authorization
  • Information exchange
  • API security
  • GraphQL authentication
Security Note: JWTs should always be verified server-side. Never trust JWT contents without proper signature verification.

GraphQL Security

In GraphQL introspection testing, JWT tokens are often used for authentication. Analyzing JWT contents can reveal user permissions and sensitive information about the GraphQL schema.
