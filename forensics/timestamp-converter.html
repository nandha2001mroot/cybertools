About Timestamps

Timestamps are used to record when events occurred in computer systems. Different systems use different timestamp formats, requiring conversion for analysis.

Common Timestamp Formats:
  • Unix Timestamp: Seconds since Jan 1, 1970 (UTC)
  • Windows FILETIME: 100-nanosecond intervals since Jan 1, 1601 (UTC)
  • ISO 8601: Standardized date/time format (YYYY-MM-DDTHH:MM:SS)
  • RFC 2822: Email date format (Day, DD Mon YYYY HH:MM:SS TZ)
