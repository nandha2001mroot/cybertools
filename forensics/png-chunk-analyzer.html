About PNG Chunks

PNG images are composed of chunks containing different types of data. Each chunk has a specific purpose and can contain metadata or hidden information.

Common PNG Chunks:
  • IHDR: Image header with dimensions and properties
  • PLTE: Palette for indexed-color images
  • IDAT: Image data chunks
  • IEND: Image trailer marker
  • tEXt: Textual data
  • zTXt: Compressed textual data
  • iTXt: International textual data

Digital Forensics

In digital forensics, PNG chunk analysis is crucial for detecting hidden data and metadata in images. Nandha Kumar M's research shows that steganography techniques often utilize ancillary PNG chunks to conceal information.

Malware Analysis

PNG chunk analysis is important for malware analysis when examining suspicious images that may contain embedded malicious payloads.
