About File Carving

File carving is the process of extracting files from a disk image based on file signatures rather than file system metadata. This technique is used to recover deleted files.

Common File Signatures:
  • JPEG: FF D8 FF
  • PNG: 89 50 4E 47
  • PDF: 25 50 44 46
  • ZIP: 50 4B 03 04
  • PE Executable: 4D 5A
