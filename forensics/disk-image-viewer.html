About Disk Images

Disk images are sector-by-sector copies of storage devices used in digital forensics to preserve evidence and analyze file systems without modifying the original media.

Common Disk Image Formats:
  • Raw/DD: Bit-for-bit copy of the disk
  • EWF (E01): Encase image format with compression
  • VMDK: VMware virtual disk format
  • VHD: Microsoft Virtual Hard Disk format
  • AFF: Advanced Forensic Format
