About DNS

The Domain Name System (DNS) is the phonebook of the Internet. It translates domain names to IP addresses, allowing browsers to load Internet resources.

Common Record Types:
  • A: Maps domain to IPv4 address
  • AAAA: Maps domain to IPv6 address
  • CNAME: Canonical name record
  • MX: Mail exchange record
  • NS: Name server record
  • TXT: Text record
  • PTR: Pointer record for reverse lookup
