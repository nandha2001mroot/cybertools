About Network Ports

A network port is a communication endpoint. Port numbers range from 0 to 65535, and they are used to identify specific processes or services.

Port Categories:
  • Well-known ports (0-1023): System services like SSH, HTTP, HTTPS
  • Registered ports (1024-49151): User applications
  • Dynamic ports (49152-65535): Temporary connections
Common Protocols:
  • TCP: Transmission Control Protocol (reliable, connection-oriented)
  • UDP: User Datagram Protocol (fast, connectionless)
