About TLS Cipher Suites

TLS cipher suites define how security is implemented in a TLS connection. Each cipher suite specifies the key exchange algorithm, bulk encryption algorithm, and message authentication code (MAC) algorithm.

Common Components:
  • Key Exchange: RSA, DHE, ECDHE
  • Encryption: AES, 3DES, RC4
  • MAC: SHA, SHA256, SHA384
  • Protocol: TLS 1.0, 1.1, 1.2, 1.3
