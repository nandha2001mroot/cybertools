About One-Time Pad

The One-Time Pad (OTP) is an encryption technique that cannot be cracked if used correctly. It requires the key to be truly random, at least as long as the plaintext, never reused in whole or part, and kept completely secret.

Characteristics:
  • Theoretically unbreakable encryption
  • Requires key of same length as plaintext
  • Key must be used only once
  • Perfect secrecy when implemented correctly
