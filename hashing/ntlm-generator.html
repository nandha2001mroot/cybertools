About NTLM

NTLM (NT LAN Manager) is a suite of Microsoft security protocols that authenticates users. NTLM is the successor to the authentication protocol in Microsoft LAN Manager (LANMAN), an older Microsoft product, and attempts to provide backwards compatibility with LANMAN.

Characteristics:
  • Used in Windows authentication
  • Based on MD4 hash function
  • Considered weak by modern standards
  • Being replaced by Kerberos
  • Still used in legacy systems
Common Use Cases:
  • Windows domain authentication
  • Legacy system compatibility
  • Penetration testing
  • Security research
  • Hash comparison in security tools
Security Warning: NTLM is considered insecure by modern standards. Use Kerberos or other stronger authentication methods when possible.
