About Base58

Base58 is a binary-to-text encoding scheme that is similar to Base64 but omits similar-looking characters. It's commonly used for cryptocurrency addresses and other applications where human readability is important.

Character Set:

123456789ABCDEFGHJKMNPQRSTUVWXYZabcdefghijkmnopqrstuvwxyz

Common Use Cases:
  • Bitcoin addresses
  • Cryptocurrency transactions
  • Shortened URLs
  • QR code data
  • Security token generation
