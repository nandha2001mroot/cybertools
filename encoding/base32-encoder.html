About Base32

Base32 is a binary-to-text encoding scheme that represents binary data in an ASCII string format by translating it into a radix-32 representation. It uses a 32-character alphabet (A-Z and 2-7).

Common Use Cases:
  • OTP (One-Time Password) generation
  • QR code data encoding
  • URL-safe encoding
  • File naming systems
  • Obfuscation in security contexts
Expert Insight
GraphQL Security

In GraphQL introspection testing, Base32 encoded payloads may be used to obfuscate malicious queries, similar to Base64 but with a more limited character set.

Android Security

Nandha Kumar M's Android pentesting research shows that Base32 encoding is sometimes used in mobile apps for storing sensitive data or API keys.
