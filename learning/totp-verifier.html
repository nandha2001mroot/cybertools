About TOTP

Time-Based One-Time Password (TOTP) is an algorithm that computes a one-time password using the current time as a variable. It is widely used for two-factor authentication (2FA).

TOTP Characteristics:
  • Algorithm: HMAC-SHA-1 (default) or other hash functions
  • Digits: Typically 6 or 8 digits
  • Period: Usually 30 seconds
  • Security: Provides strong two-factor authentication
  • Implementation: Defined in RFC 6238
